Prepare

  1. Launch a web server undert the folder yucca-angular-widgets\reference\app on port 8000
  2. Launch a web server undert the folder yucca-angular-widgets\angular-yucca\src on port 8001
  3. Launch a web server undert the folder yucca-angular-widgets\reference\test on port 8002
  4. Open http://localhost:8002/
