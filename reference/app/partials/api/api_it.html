Metadata API
Cercare Stream e Dataset tramite i metadati

Filtri di ricerca

Esposizione API di ricerca e dettaglio dei Metadati di stream e dataset tramite ricerca libera e/o filtri specifici su area di lavoro, Organizzazione, Ambito, tag, visibilità

Formati di output

I risultati delle query possono essere in questi formati:

  • Json
  • CKAN
  • Data Catalog Interoperability Protocol DCAT-AP-IT in formato JSON+LD

Esempio di chiamata

Ricerca dataset o stream relativi all'acqua 
https://api.smartdatanet.it/metadataapi/api/v02/search?q=acqua
Risposta (parziale)
{
  "start": 0,
  "count": 12,
  "totalCount": 23,
  "totalPages": 2,
  "params": {
    "q": "acqua"
  },
  "metadata": [
    {
      "name": "Patrimonio immobiliare regionale - Beni demaniali",
      "version": "5",
       …

 

Data API
Query sui dati

Interrogare la piattaforma e filtrare i dati

Esposizione dei dati tramite servizi rest-http implementati secondo le specifiche del protocollo oData

Si possono effettuare query paginate con filtri su tutte le colonne del dataset.Filtri di tipo numerico, temporale o sul contenuto del testo in base alla tipologia di dato

Formati di output

I risultati delle query possono essere in questi formati:

  • Json
  • XML
