Installation through Bower

You don't know Bower? Discover how to use it

Bower is a tool that lets you download the frontend resources for html / javascript projects saving them directly into the project. Bower working through the console of NodeJS and use Git for download the sources

Installation

  1. Download and installNodeJS
  2. Download and install the Git client
  3. Open the command prompt of NodeJS and install Bowetr running the command 
    npm install -g bower
  4. To install the project resource use the command prompt of NodeJS and run 
    bower install nome_libreria
    from the root of the project
$ bower install yucca-angular-widgets

Bower automatically install all the dependencies . If it won't for some reason, install it manually: 

$ bower install angular
$ bower install stomp-websocket
$ bower install d3
$ bower install nvd3
$ bower install angular-nvd3
$ bower install angular-leaflet-directive

Adding dependency

In the section head of the main site of tour portal, add 

<meta charset="utf-8">  <!-- it's important for d3.js -->
<link rel="stylesheet" href="bower_components/nvd3/build/nv.d3.min.css">
<link rel="stylesheet" href="bower_components/leaflet/dist/leaflet.css">
<link rel="stylesheet" href="bower_components/yucca-angular-widgets/dist/style/angular-yucca.css">

<script type="text/javascript" src="bower_components/d3/d3.js"></script>
<script type="text/javascript" src="bower_components/nvd3/build/nv.d3.min.js"></script>
<script type="text/javascript" src="bower_components/leaflet/dist/leaflet.js"></script>

<script type="text/javascript" src="bower_components/angular/angular.js"></script>
<script type="text/javascript" src="bower_components/angular-sanitize/angular-sanitize.min.js"></script>
<script type="text/javascript" src="bower_components/angular-nvd3/dist/angular-nvd3.min.js"></script>
<script type="text/javascript" src="bower_components/stomp-websocket/lib/stomp.min.js"></script>
<script type="text/javascript" src="bower_components/angular-leaflet-directive/dist/angular-leaflet-directive.min.js"></script>
<script type="text/javascript" src="bower_components/yucca-angular-widgets/dist/js/angular-yucca.js"></script>

Use

Add the modules yucca.plugin and ngSanitize in your angular module 

angular.module('myApp', ['yucca.plugin']).controller('myCtrl', function($scope){
	//...  You  eventual business logic
})

In the html file you can use directlu the widgets 

<div ng-app="myApp">
  <div ng-controller="myCtrl">
    …
    <ng-yucca-dataset-image-gallery tenant_code='sandbox' dataset_code='Fotoagro_576'  />
    …
  </div>
</div>
You can use the widgets even if the sites do not use AngularJS: the panel that contain the widgetbecomes a mini app AngularJS.
