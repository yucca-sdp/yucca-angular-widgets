Installazione

E' possibile effetture il download della libreria tramite Bower, oppure direttamente dai sorgenti su Github, o partire da questo progetto minimale che contiene le librerie scaricate e una pagina html con due widget di esempio

Installazione tramite Bower

Non conosci Bower? Scopri come usarlo

Bower à un tool che permette di scaricare le risorse di frontend per progetti html/javascript salvandole direttamente nel progetto stesso. Bower funziona tramite la console di NodeJS e ha bisogno di un client Git per scaricare i sorgenti

Installazione

  1. Scarica e installa NodeJS
  2. Scarica e installa il client Git
  3. Apri la console di comandi di NodeJS e installa Bower lanciando il comando 
    npm install -g bower
  4. Le risorse si installano da console, posizionandosi nella root del progetto da aggiornare e lanciando il comando 
    bower install nome_libreria
$ bower install yucca-angular-widgets

Bower dovrebbe installare in automatico tutte le dipendenze, nel caso non lo facesse

$ bower install angular
$ bower install stomp-websocket
$ bower install d3
$ bower install nvd3
$ bower install angular-nvd3
$ bower install angular-leaflet-directive

Aggiunta delle dipendenze

Nel sezione head della pagina principale del tuo portale inserire 

<meta charset="utf-8">  <!-- it's important for d3.js -->
<link rel="stylesheet" href="bower_components/nvd3/build/nv.d3.min.css">
<link rel="stylesheet" href="bower_components/leaflet/dist/leaflet.css">
<link rel="stylesheet" href="bower_components/yucca-angular-widgets/dist/style/angular-yucca.css">

<script type="text/javascript" src="bower_components/d3/d3.js"></script>
<script type="text/javascript" src="bower_components/nvd3/build/nv.d3.min.js"></script>
<script type="text/javascript" src="bower_components/leaflet/dist/leaflet.js"></script>

<script type="text/javascript" src="bower_components/angular/angular.js"></script>
<script type="text/javascript" src="bower_components/angular-sanitize/angular-sanitize.min.js"></script>
<script type="text/javascript" src="bower_components/angular-nvd3/dist/angular-nvd3.min.js"></script>
<script type="text/javascript" src="bower_components/stomp-websocket/lib/stomp.min.js"></script>
<script type="text/javascript" src="bower_components/angular-leaflet-directive/dist/angular-leaflet-directive.min.js"></script>
<script type="text/javascript" src="bower_components/yucca-angular-widgets/dist/js/angular-yucca.js"></script>

Utilizzo

Aggiungi i moduli yucca.plugin e ngSanitize nel tuo modulo angular 

angular.module('myApp', ['yucca.plugin', 'ngSanitize']).controller('myCtrl', function($scope){
	//...  la tua eventuale logica di business
})

Nei file html potrai usare direttamente i widget 

<div ng-app="myApp">
  <div ng-controller="myCtrl">
    …
    <ng-yucca-dataset-image-gallery tenant_code='sandbox' dataset_code='Fotoagro_576'  />
    …
  </div>
</div>
Si possono usare i widgets anche se i siti non utilizzano AngularJS: il pannello che conterrà il widget diventa una mini app AngularJS.
